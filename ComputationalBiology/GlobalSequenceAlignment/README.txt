1: How to run application
   a: In terminal type: java -jar Zalgorithm.jar X Y Z FASTA.txt
      X: delta(match) 
	  Y: delta(mismatch) 
	  Z: delta(insertion/deletion) <MUST BE NEGATIV INTEGER>
   b: FASTA.txt should include your strings;
      S: on first line
	  T: on second line
2: Demonstration on YouTube: https://youtu.be/2RjbHi6S6_c
